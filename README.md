# NeuraGenesis

NeuraGenesis is a welfare-first developmental intelligence scaffold built around a parent-guided AI growth model.

Current foundation:
- Expo / React Native parent control app
- Supabase auth, data, realtime, and edge-function boundaries
- approval-gated developmental milestones
- welfare-first learning and embodiment controls
- Dot Matrix perception groundwork for future camera and LiDAR fusion
- simulation-first validation before risky capability changes

Core operator workflow:
- sign in as parent
- review child profiles
- monitor welfare, incidents, and pending approvals
- approve or deny milestone progression
- require simulation success before embodiment

Documentation lives in `/docs`, including the Pages entry point at `docs/index.html`.
